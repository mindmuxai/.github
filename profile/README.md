# MindMux

> A local-first project brain for builders and agents.

## What we do

MindMux is a local-first AI workbench that turns project conversations into structured knowledge.

**Chat → Brain → Task**

- **Chat** with AI about architecture, decisions, and trade-offs.
- **Brain** stores the distilled knowledge as plain Markdown in a `brain/` folder.
- **Task** hands off executable work to agents and tools — with the full context attached.

Your project knowledge lives on your local machine: portable, diffable, and owned by you.

## Why

Every new chat and agent starts from zero context. Product decisions, technical reasoning, and rejected options get lost across docs, issues, and threads. MindMux keeps the context in one structured, portable place.

## Openness

We are progressively open-sourcing the Project Brain Standard and the MindMux client. Knowledge should be portable and owned by the team that creates it.
